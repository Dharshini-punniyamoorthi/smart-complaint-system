# Modules

## 1. User Module
- Register and login
- Submit complaints with category and description
- Track complaint status (New, In Progress, On Hold, Resolved)
- View complaint history

## 2. Staff Module
- View complaints assigned to them
- Update complaint status
- Comment or respond to complaints

## 3. Admin Module
- Manage users and staff accounts
- Monitor all complaints
- Generate reports and analytics
- Ensure smooth workflow and accountability
